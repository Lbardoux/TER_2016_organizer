 5  Un module remplit de fonctionnalités communes à l'application.

 7  Fonctionnalités

 9   
10  Voici les fonctionnalités offertes : 
11          1. L{enum.enum} qui permet de créer des valeurs énumérées. 
12                   
13                  Cela permet de créer un enum comme dans d'autre langages. 
14                  Merci à Alec Thomas de StackOverflow pour la solution. 
15                   
16                  http://stackoverflow.com/questions/36932/how-can-i-represent-an-enum-in-python 
17                   
18                  >>> variable = enum("VALEUR1", 'VALEUR2', 'VALEUR3') 
19                  # on a alors variable.VALEUR1, variable.VALEUR2, etc 
20                   
21          2. L{extension.nomFichier} qui permet de récupérer l'extension d'un nom de fichier. 
22                   
23                  >>> nomFichier("/home/lol/fichier.drole.ics") 
24                  ics 
25                   
26          3. L{Fabrique} est une classe générique qui encapsule les fonctions essentielles d'une fabrique. 
27                   
28                  Il suffit d'en dériver en fournissant un dictionnaire lors de la construction. 
29                  Et pour plus de simplicité, mixer les clefs avec un L{enum} ! 
30                   
31                  >>> class MaFabrique(Fabrique): 
32                  ...     def __init__(self): 
33                  ...             dico = { 
34                  ...                     monEnum.CLEF1 : lambda x : x, 
35                  ...                     monEnum.CLEF2 : fonction 
36                  ...             } 
37                  ...             Fabrique.__init__(self, dico) 
38                   
39                  Il suffit ensuite d'appeler la méthode L{Fabrique.Fabrique.fabrique} avec les bons arguments. 
40                   
41                  >>> mf = MaFabrique() 
42                  >>> mf.fabrique(argument_voulu) 
43                  # résultat de la construction 
44           
45          4. L{GenerateurId} qui est un générateur générique d'objet. 
46                   
47                  Il suffit de lui préciser lors de la construction le point de départ, 
48                  B{inclus dans la génération} et une fonction/lambda qui se chargera de l'opération 
49                  d'incrément. 
50                   
51                  >>> monGenerateur = GenerateurId(0, lambda x : x+2) 
52                  >>> monGenerateur.suivant() 
53                  0 
54                  >>> monGenerateur.suivant() 
55                  2 
56           
57          5. L{VerifierContenuListe.verifier} qui permet de vérifier le contenu d'une liste. 
58                   
59                  Il suffit de lui préciser un conteneur itérable (qui accepte le format B{for element in conteneur:}, 
60                  et une fonction/lambda qui effectuera les tests sur chaque élément. 
61                  La fonction renvoie True si tous les tests passent, False sinon. 
62                   
63                  >>> verifier([1, 2, 3, "oups", 4], lambda x : type(x) is int) 
64                  False 
65                   
66          6. L{observateur} qui est un package contenant le I{design pattern} Observateur. 
67   
68  @author: Laurent Bardoux p1108365
