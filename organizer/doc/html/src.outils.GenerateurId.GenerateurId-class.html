Class GenerateurId

source code

object --+

        GenerateurId

Une classe qui va se charger de générer des identifiants. Ces identifiants devant etre unique, un générateur comme celui-ci sera communs à tous les Agendas lors du déroulement. Son role est donc d'assurer l'unicité des identifiants produits.

Initialise ce générateur avec comme valeur initiale pointDeDepart. L'incrément de cette valeur initiale sera réalisé par la lambdaIncrement. Cette lambda doit prendre un argument et renvoyer le suivant en fonction de celui-ci.

Parameters:
  • self - L'argument implicite.
  • pointDeDepart (ce qu'on veut.) - le commencement de l'incrément.
  • lambdaIncrement (lambda expression renvoyant le meme type que pointDeDepart.) - une fonction qui sera utilisé pour réalisé l'incrément.
