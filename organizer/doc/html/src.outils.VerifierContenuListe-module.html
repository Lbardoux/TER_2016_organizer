verifier(conteneur, fonctionBooleenne)

source code 

Cette fonction utilitaire va permettre de vérifier le contenu d'un conteneur supportant le for in. Sur chaque élément du conteneur, fonctionBooleenne ser appelée.

Parameters:
  • conteneur - un conteneur supportant la boucle for @type : conteneur(__iter__), list, dict, tuple
  • fonctionBooleenne (une fonction/lambda renvoyant true si l'élément est OK @return : true si tout est OK, false sinon @author : Laurent Bardoux p1108365) - ce qui va tester.
