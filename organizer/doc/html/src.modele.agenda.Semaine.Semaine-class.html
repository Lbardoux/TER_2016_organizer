Class Semaine

source code

       object --+    
                |    
Modifier.Modifier --+
                    |
                   Semaine

La classe qui va définir une Semaine. Il faut savoir qu'une semaine d'un mois peut très bien contenir moins de 7 jours. Par exemple, elle peut se finir un jeudi (passage au mois d'après, dont la première semaine commence un vendredi)

Author: Laurent Bardoux p1108365

Version: 1.0

ajouterCreneau(self, jour, debut, fin, typeCreneau=0)

source code 

Etape 4 de la descente dans l'architecture. Ceci va "ajouter" un Creneau dans le jour, entre debut et fin.

Parameters:
  • self - L'argument implicite
  • jour (int) - le numéro du jour dans lequel insérer ce créneau.
  • debut (int [1, 48]) - l'heure de début du créneau
  • fin (int [1, 48]) - l'heure de fin du créneau
  • typeCreneau (enum) - une valeur enumérée pour la fabrique de creneau
Returns: Creneau
un Creneau valable.
Raises:
  • ValueError - Si un des arguments est erroné.

Precondition: debut < fin, debut/fin doivent etre compris dans leurs intervalles respectifs
