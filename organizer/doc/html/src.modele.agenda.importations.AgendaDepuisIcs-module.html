importer(agenda, nom)

source code 

Cette fonction est étroitement liée à un Agenda pour l'import depuis un fichier ICS. Il utilisera aveuglément les méthodes d'insertions proposées par la partie frontale du modèle.

Parameters:
  • agenda (Agenda) - Un Agenda à remplir/compléter.
  • nom (str) - un nom de fichier
Raises:
  • ValueError - si le parsing marche mal (erreur de syntaxe en général).
  • IOError - si le fichier est introuvable

Precondition: le nom doit avoir .ics comme extension !

Postcondition: l'Agenda est remplis avec les champs valides.
