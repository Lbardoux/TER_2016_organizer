avoirPremierCreneauDepuis(liste, depart)

source code 

Cette fonction permet de récupérer le premier "creneau" au format numérique pour avoir un point de départ. L'idée est de chercher le premier creneau occupé (en ignorant les blancs dans la liste) à partir de départ.

Parameters:
  • liste (list) - Une liste de Creneau.
  • depart (int) - le point de départ de la recherche
Returns: tuple
la valeur de debut et de fin tel que debut >= depart
Raises:
  • IndexError - Si aucun Creneau n'a été trouvé.

convertitListe(liste)

source code 

On apporte un peu de connaissance (oui c'est mal ><). on transforme une liste de Creneaux en une liste de 49 éléments contenant soit un créneau, soit None. On pourra alors parcourir séquentiellement cette liste, avec None = pas de créneau, et une liste de référence = un/des Creneau(x).

Parameters:
  • liste (list(Creneau)) - La liste des Creneaux à convertir.
Returns: list
la liste convertit.
