Class Observable

source code

object --+
         |
        Observable

L'alter-ego de la classe Observeur. Elle offre et implémente tout le nécessaire à l'emploi du design pattern observateur. A l'inverse de Observeur, inutile de surcharger les fonctions dans son cas. En complément, elle offre un decorator, permettant de tagger les cas ou l'on doit notifier (mais sans propager d'arguments, hélas)

Author: Laurent Bardoux p1108365
