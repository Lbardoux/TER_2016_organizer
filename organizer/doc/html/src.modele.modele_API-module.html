Module modele_API

source code

Ce module contient la classe qui sert de partie frontale pour le modèle.

Elle aura pour principale tache le chargement/déchargement de fichiers, et les fonctionnalités imposant le recours à plusieurs Agenda différents (Comme le diff, etc).

Utilisation

Il suffit juste d'instancier un objet de type ModeleAgenda.

>>> monModele = ModeleAgenda()
# On peut donc utiliser monModele ensuite

Fonctions publiques

  1. ModeleAgenda.chargerAgenda(self, nomFichier)

    Description:

    Cette fonction permet de charger un agenda à partir d'un fichier .ics

    Usage

    >>> monModele.chargerAgenda("monFichier.ics")
<Agenda 0x********>
  2. ModeleAgenda.dechargerAgenda(self, agenda)

    Description:

    Cette fonction permet de décharger un agenda à partir de son nom

    Usage

    >>> monModele.dechargerAgenda(monAgenda.nom)
# Suppression en mémoire de cet Agenda si il est chargé.
  3. ModeleAgenda.ajouterDependanceA(self, agenda, nomFichier)

    Description:

    Cette fonction permet d'ajouter une dépendance à agenda en chargeant un nouvel agenda depuis nomFichier.

    Usage

    >>> monModele.ajouterDependanceA(monAgenda, "monAgendaDependance.ics")
# monAgenda possède désormais monAgendaDependance comme ... dépendance.
  4. ModeleAgenda.enleverDependanceDe(self, agenda, nomDependance)

    Description:

    Cette fonction permet d'enlever une dépendance à agenda en se basant sur le nom de la dépendance nomDependance.

    Usage

    >>> monModele.enleverDependanceDe(monAgenda, "monAgendaDependance.ics")
# Si elle existe, cette dépendance (et toutes les dépendances liées) seront supprimés.
  5. ModeleAgenda.sauvegarderAgenda(self, agenda, nomDeSauvegarde=None)

    Description:

    Cette fonction permet de sauvegarder un agenda. Si aucun nom n'est fourni, cela équivaut à sauvegarder avec le nom de l'agenda. Si nomDeSauvegarde est donné, cela équivaut à un enregistrer sous.

    Usage

    >>> monModele.sauvegarderAgenda(monAgenda)
# monAgenda est sauvegarder avec comme nom I{monAgenda.nom}
    >>> monModele.sauvegarderAgenda(monAgenda, "nouveauNom.ics")
# monAgenda est sauvegarder avec comme nom nouveauNom.ics
