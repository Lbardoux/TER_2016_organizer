Class Annee

source code

       object --+    
                |    
Modifier.Modifier --+
                    |
                   Annee

La classe la plus haute dans la hiérarchie de stockage d'un agenda. Elle représente une année, du 1er janvier au 31 decembre. De ce fait, elle gère les années spécifiques, et l'arborescence des Mois, Semaine et Jour. C'est également le point d'entrée de la structure de stockage.

Author: Laurent Bardoux p1108365

Version: 1.0
