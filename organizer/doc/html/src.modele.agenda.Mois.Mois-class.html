ajouterCreneau(self, jour, debut, fin, typeCreneau=0)

source code 

Etape 3 de la descente dans l'architecture. Ceci va "ajouter" un Creneau dans le jour, entre debut et fin.

Parameters:
  • self - L'argument implicite
  • jour (int) - le numéro du jour dans lequel insérer ce créneau.
  • debut (int [1, 48]) - l'heure de début du créneau
  • fin (int [1, 48]) - l'heure de fin du créneau
  • typeCreneau (enum) - une valeur enumérée pour la fabrique de creneau
Returns: Creneau
un Creneau
Raises:
  • ValueError - SI jamais un des arguments ne permet pas la création d'un Creneau.

Precondition: debut < fin, jour/debut/fin doivent etre compris dans leurs intervalles respectifs
