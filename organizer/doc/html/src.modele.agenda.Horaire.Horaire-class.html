Class Horaire

source code

object --+
         |
        Horaire

La classe qui définit un horaire pour un Créneau. Notre calendrier est découpé en semaine (allant de Lundi à Vendredi). Ces jours sont découpées en quart d'heures de 8h du matin à 20h. Donc, pour modéliser une semaine, nous nous basons sur le nombre de quart d'heures disponibles, soit : 12_heures * 4 = 240 quart d'heures Il ne nous reste qu'a faire la division euclidienne du numéro par 240 pour connaitre la semaine associée.

Author: Laurent Bardoux p1108365

Version: 1.0
