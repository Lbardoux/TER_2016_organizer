Class Creneau

source code

object --+
         |
        Creneau

La classe centrale d'un agenda. Le créneau est un emplacement que l'on alloue sur l'agenda. Il contient donc parfois de l'information supplémentaire.

Author: Laurent Bardoux p1108365

Version: 1.0

ajouterInformation(self, clef, info)

source code 

Permet d'ajouter une information dans le dictionnaire, sous la forme clef -> info. Vous pouvez vérifier l'unicité d'une clef grâce à la méthode self.existe. Si la clef existe déjà, rien ne se passera (pas d'écrasement).

Parameters:
  • self - l'argument implicite.
  • clef (str) - la clef pour identifier l'information dans le dictionnaire.
  • info (object) - Ce que l'on veut stocker dans le dictionnaire.

Precondition: type(clef) is str
